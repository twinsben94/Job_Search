# NAVER Cloud — 지원 정보 / 채용 / 연락처
> 조사일: 2026-06-10 · 공고는 수시로 변동되므로 지원 전 채용 페이지 재확인 필수

## 1. 적합도 한 줄
지도/공간데이터 이해 + AI 플랫폼·B2B 솔루션 사업화 접점. NAVER LABS가 '기술 깊이'면 NAVER Cloud는 '제품·사업화'. (적합도: 매우 높음)

## 2. 지원해볼 만한 포지션 (직무군)
2026년 다수 공고 진행 중(AI 직무 활발). 관심 직무군:
- **Maps API / Location-based Service (LBS)** 관련 AI·플랫폼
- **Cloud AI Platform / B2B AI Solution** (기획·개발)
- **Digital Twin / Robotics / AI DX** 솔루션
- Data Scientist / AI Engineer (HyperCLOVA X 등 AI 플랫폼)

## 3. 채용 페이지 / 인재풀
- 공식 채용: https://recruit.navercloudcorp.com/
- 진행 공고(집계 사이트): 사람인·잡코리아·인크루트 "네이버클라우드(주)"
- 제품/서비스: https://www.ncloud.com/

## 4. 직접 지원(CV/포트폴리오) 연락처
- 공식 채용 사이트 내 **상시지원/인재풀** 활용 (별도 공개 채용메일은 미확인 — recruit 페이지 문의 채널 사용)
- 위치: 분당/수도권 계열

## 5. 지원 팁
- 포지셔닝: 지도·공간데이터 활용 + **AI 플랫폼/솔루션 기획·구현** 역량.
- 연결 포인트: 위성·GIS·street-view 데이터 파이프라인 → Maps/LBS 데이터 품질·신규 피처, GEE 기반 대규모 처리 경험 → Cloud AI.
- 보완: API 서빙, Docker, 모델 모니터링 등 MLOps 키워드.
