# 국회미래연구원 (NAFI) — 지원 정보 / 채용 / 연락처
> 조사일: 2026-06-10 · 공고는 수시로 변동되므로 지원 전 채용 페이지 재확인 필수

## 1. 적합도 한 줄
산업계는 아니지만 박사 연구의 **공공적 의미**를 가장 잘 살림. 기후·농업·AI 기반 국가전략 연구와 연결. (적합도: 중상)

## 2. 지원해볼 만한 포지션 (확인된 공고/직무)
- **연구직(부연구위원 / 위촉연구원)** ← 박사 학위자 대상
  - 예: 2026년도 제2차 연구직(부연구위원) 모집(2026-03-31~04-22 진행 사례)
  - 관심 분야: 기후환경, AI/데이터 기반 미래예측, 농업/식량안보 정책연구
- 연구는 분야별 공모로 진행되므로 **기후·환경·과학기술** 공고를 주기적으로 확인

## 3. 채용 페이지 / 인재풀
- 공식 채용공고: https://www.nafi.re.kr/new/job.do
- 채용 사이트: https://nafi.saramin.co.kr/ · https://recruit.incruit.com/nafi/
- 위치: 서울 영등포구 의사당대로 1, 국회도서관 5층 (여의도)

## 4. 직접 지원(CV/포트폴리오)
- nafi.re.kr 공고별 지원(공공기관 — 공고 기간 내 정식 접수). 상시 송부보다 **공고 모니터링**이 핵심.

## 5. 지원 팁
- 선택 기준: 높은 연봉/기업 커리어보다 **정책 영향력·연구 지속성**을 원할 때.
- 포지셔닝: 기후·농업·AI 기반 국가전략 연구자. 정책 issue brief·시나리오 작성 역량 보완.
